# TI-84-Plus-Ce-Python-Programs
This contains two files, one that solves Quadratics Equations instantly. 

The other is a game called "Guess my number", this was a small project in order to get better at programming.  

In order to transfer files, to the calculator, use the software TI-connect-Ce. 

This software is located on the Texas Instruments website. 

From there you will need to transfer the files to the calculator. 

There all plenty of tutorials online that tell you how to do it.
